# スマレジ在庫管理API調査結果

## 発見

左メニューに「在庫」セクションを発見！

### 在庫管理API（Inventory）のエンドポイント

| エンドポイント | メソッド | 説明 |
|:---|:---|:---|
| `/stock` | GET | **在庫一覧取得** |
| `/stock` | PATCH | 在庫更新 |

## エンドポイントURL

- **サンドボックス**: `https://api.smaregi.dev/{contract_id}/pos`
- **本番**: `https://api.smaregi.jp/{contract_id}/pos`

## 重要な発見

**在庫管理APIも同じ `/pos` パスを使用している！**

つまり、POS APIと在庫管理APIは同じエンドポイントを共有している。
問題は `/stock` エンドポイントの使い方にある可能性が高い。
