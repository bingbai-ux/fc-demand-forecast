# UI Issues Analysis

## スクリーンショット分析結果

### 問題1: グループヘッダーと列ヘッダーのずれ
- スクリーンショット（21.31.41.png）を見ると、グループヘッダーの順序は：
  - 商品情報 → 在庫 → 売上数 → 集計 → 操作
- しかし、実際の列ヘッダーの順序は：
  - 商品名、ブランド、カテゴリ、仕入先、単価、原価（商品情報）
  - 在庫（在庫）
  - 合計数、売上合計、原価合計、粗利率、平均売上、平均売数（集計）
  - 12/22週、12/29週、1/5週、1/12週、1/19週（売上数）
  - 操作

**問題**: グループヘッダーの「売上数」と「集計」の順序が逆になっている

### 問題2: グラフの左側の指標
- スクリーンショット（21.31.56.png）を見ると、インフォカードには：
  - 合計売上数: 58個
  - 日平均: 8.3個
  - 売上金額: ¥22,726
  - 粗利率: 87.8%
- 「平均売上金額」が表示されていない

**修正**: 「粗利率」を「平均売上金額」に変更する必要がある

### 問題3: 店舗選択と期間の配置
- スクリーンショット（21.33.10.png）を見ると：
  - 店舗選択: 別の行に配置されている
  - 期間選択: 別の行に配置されている
- これらを同じ行に統合する必要がある

### 問題4: グラフのツールチップ
- スクリーンショット（21.33.31.png, 21.38.03.png）を見ると：
  - グラフには「平均」の点線が表示されている
  - しかし、ホバー時にツールチップが表示されていない

**修正**: Rechartsのツールチップを有効にする必要がある

## 修正計画

1. DataTable.tsx: グループヘッダーの順序を修正
   - 「商品情報」→「在庫」→「集計」→「売上数」→「操作」の順序に変更

2. App.tsx: インフォカードの「粗利率」を「平均売上金額」に変更

3. App.tsx: 店舗選択と期間選択を同じ行に統合

4. InlineChart.tsx: ツールチップを有効にして売上数を表示
