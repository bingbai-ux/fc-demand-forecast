# 発注書PDFテンプレート仕様

## 概要
FOOD&COMPANY需要予測システムで使用する発注書PDFのレイアウト仕様です。

## ページ設定
- サイズ: A4
- マージン: 50pt
- フォント: NotoSansJP-Regular（日本語対応）

## レイアウト構成

### 1. ヘッダー部（右上）
- **位置**: X=380, Y=40
- **内容**:
  - 発注日: YYYY-MM-DD
  - 発注番号: ORD-YYYYMMDD-XXX

### 2. タイトル
- **位置**: 中央, Y=85
- **フォントサイズ**: 24pt
- **テキスト**: 「発注書」
- **下線**: Y=120（タイトルとの間に5ptのスペース）

### 3. 宛名（左側）
- **位置**: X=50, Y=140
- **内容**:
  - 会社名（14pt）
  - 「御中」（10pt）

### 4. 発注者情報（右側）
- **位置**: X=350, Y=140
- **フォントサイズ**: 9pt
- **内容**:
  - 株式会社FOOD&COMPANY
  - 店舗名
  - TEL: 電話番号
  - 担当: 担当者名

### 5. 挨拶文
- **位置**: X=50, Y=210
- **フォントサイズ**: 9pt
- **テキスト**: 「平素は格別のお引き立てを賜り、厚く御礼申し上げます。下記の通り発注いたしますので、ご手配のほどよろしくお願い申し上げます。」

### 6. 納品希望日
- **位置**: Y=240
- **ラベル**: 黒背景（80x20pt）、白文字
- **値**: 日付（12pt）

### 7. お届け先情報
- **位置**: Y=280
- **ラベル**: 黒背景（70x18pt）、白文字「お届け先」
- **内容**（9pt）:
  - (株)FOOD&COMPANY 店舗名
  - 〒郵便番号
  - 住所
  - TEL: 電話番号
  - 担当: 担当者名

### 8. 明細テーブル
- **位置**: Y=400
- **列幅**:
  - No.: 35pt
  - ブランド: 100pt
  - 商品名: 280pt
  - 数量: 50pt
- **ヘッダー**: 黒背景、白文字
- **行**: 交互に背景色（#f5f5f5）

### 9. 合計数量
- **位置**: テーブル下 + 15pt
- **フォントサイズ**: 11pt
- **内容**: 「合計数量: XX 個」

### 10. 備考（オプション）
- **位置**: 合計数量下 + 40pt
- **内容**: 【備考】+ 備考テキスト

### 11. フッター
- **位置**: Y=780
- **フォントサイズ**: 8pt
- **色**: #999999
- **テキスト**: 「この発注書はFOOD&COMPANY需要予測システムから発行されました」

## 使用変数

| 変数名 | 説明 | 例 |
|--------|------|-----|
| order_number | 発注番号 | ORD-20260127-001 |
| order_date | 発注日 | 2026-01-27 |
| expected_arrival | 納品希望日 | 2026-01-30 |
| display_name | 仕入先会社名 | 有限会社ノースプレインファーム |
| supplier_name | 仕入先名 | ノースプレインファーム |
| store_name | 店舗名 | FOOD&COMPANY Daikanyama T-SITE |
| postal_code | 郵便番号 | 150-0033 |
| address | 住所 | 東京都渋谷区猿楽町16-15 |
| phone | 電話番号 | 03-5990-4783 |
| contact_person | 担当者名 | Ikeda |
